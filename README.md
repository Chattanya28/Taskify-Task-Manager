# Taskify • Focus Flow

A modern, beautiful, and feature-rich Todo List web application built with **HTML, CSS, and JavaScript**.

![Taskify Preview](https://taskfytm.netlify.app/)

## ✨ Features

- **Glassmorphism Design** with smooth animations
- **Progress Circle** showing completion percentage
- **Priority Levels** (High, Medium, Low) with color indicators
- **Due Dates** for better time management
- **Categories/Tags** (Work, Personal, Health, Study, etc.)
- **Drag & Drop** reordering of tasks
- **Filter Tasks** — All, Active, Completed
- **Dark / Light Mode** toggle with persistence
- **Responsive & Mobile-friendly** design
- **Data Persistence** using LocalStorage
- Beautiful empty state illustration
- Clean, professional UI with micro-interactions
